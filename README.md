# Weatherly

## Description
Simple app that grabs phone location and displays current temperature and weather conditions.

## Skills Practiced

* Dark Mode
* Protocol/Delegate Pattern
* Closures/Completion Handlers
* HTTP Requests
* JSON Encoding/Decoding
* Grand Central Dispatch
* Core Location
